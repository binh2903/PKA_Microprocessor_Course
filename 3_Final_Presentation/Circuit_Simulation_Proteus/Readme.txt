# Proteus Project

This project was designed and simulated using Proteus Design Suite version 8.13.

## Software Requirements

To open and simulate the files in this project, you need to use Proteus Design Suite version 8.13 or higher.

**Note:** If you use a Proteus version lower than 8.13, you may encounter errors when opening the files or be unable to open them at all.

## Installation Instructions

1. Download and install Proteus Design Suite version 8.13 (or higher) from the official Labcenter Electronics website or from trusted sources.

2. Open the project file (.pdsprj) using Proteus.

3. If you encounter any errors, ensure that you are using the correct Proteus version.
